================
Related Projects
================

Build tools & task runners:

 - `make <https://www.gnu.org/software/make/>`_
 - `just <https://github.com/casey/just>`_ - command runner (no dependency tracking)
 - `invoke <https://www.pyinvoke.org/>`_ - Python task execution (no DAG/incremental builds)
 - `SCons <https://scons.org/>`_
 - `snakemake <https://snakemake.github.io/>`_ - workflow management for data analysis
 - `Rake <https://ruby.github.io/rake/>`_
 - `ant <https://ant.apache.org/>`_

There are `many <https://en.wikipedia.org/wiki/List_of_build_automation_software>`_ more...

In this `post <http://schettino72.wordpress.com/2008/04/14/doit-a-build-tool-tale/>`_ I briefly explained my motivation to start another build tool like project.
